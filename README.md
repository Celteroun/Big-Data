1 - Ingesta de datos: Se obtiene, carga y preparan los datos crudos que se usaran para todo el proyecto. Se carga el dataset desde su fuente principal.
2 - Analisis exploratorio: Analiza estadísticas descriptivas, como medias, medianas, desviación estándar, etc. Identifica outliers, valores nulos y distribuciones anómalas.
3 - Ingeniería de características: Limpia los datos a profunidad para que se puedan procesar por una máquina, óptimo para el Machine Learning.
4 - Modelado y Machine Learning: Se entrenan varios modelos de machine learning y se evalúan. Acá se materializa el objetivo del proyecto.
